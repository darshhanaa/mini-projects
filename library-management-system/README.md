# 📚 Library Management System

A simple menu-driven **Library Management System** built with Python. This project allows users to manage a collection of books by performing basic CRUD (Create, Read, Update, Delete) operations. The data is stored in a JSON file, ensuring that book records are saved even after the program is closed.

## 🚀 Features

- ➕ Add a new book
- 📖 View all books
- 🔍 Search for a book by title
- ✏️ Update book details
- ❌ Delete a book
- 💾 Save book data automatically to a JSON file
- 📂 Load existing book data when the program starts
- ✅ Input validation for a better user experience

## 🛠️ Technologies Used

- Python 3
- JSON (for data storage)

## ▶️ How to Run

1. Clone this repository or download the project files.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python library_management.py
```
